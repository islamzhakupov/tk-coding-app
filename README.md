Python-приложение (Tkinter), в котором реализованы лабораторные задания по:

энтропии и частотному анализу (source_report)
кодированию: Хаффман + Шеннон-Фано + Рид-Соломон (run_algorithms)
моделированию канала с шумом и потерями (run_channel_simulation)
RLE-сжатие/распаковка + ZIP (Lab4)
сжатие изображений (JPEG-подобное + DWT + ограничение цвета + PNG) (img_tab)

Библиотеки:
numpy
pillow
opencv-python
pywavelets
reedsolo
Базовая библиотека: tkinter
